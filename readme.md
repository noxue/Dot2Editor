## 使用文档

`Alt+1~6` 对应 h1~h6 

`Alt+B` 加粗选中的内容

`Alt+I` 插入图片（I表示`img`标签）

`Alt+A` 插入链接（A表示`a`标签）

`Alt+H` 插入横线（H表示`hr`标签）

`Alt+L` 插入线条（L表示`Line`,和`hr`的区别是，线条上方的文字会变成h2）

`Alt+T` 插入表格（T 表示`Table`）
